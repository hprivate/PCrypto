# PCrypto —— Crypto解题脚本（Python)

## RSA

### 脚本
- rsa_1.py  : 已知 `p` 、`q`、`e` 求解密文 `c`
- rsa_2.py  : 已知`p`、`q`、`dp`、`dq` 求解密文 `c`
- rsa_3.py  : RSA `共模攻击`

### 工具
1. [yafu——自适应大整数因式分解](https://sourceforge.net/projects/yafu/)
2. [公钥解析](http://www.hiencode.com/pub_asys.html)
